http-status-checker
======================

upper ruby2.0

no warranty

Usage
-------

-. mkdir target-urls 

-. written urls in target-urls/hoge.txt , target-urls/some.txt ... 

-. run `check.rb` 

-. check result `cat results.txt` 

使い方
-----------

-. target-urls というディレクトリを作成する

-. そのディレクトリとの中のファイルは適当に読むので、そこに調べたい url を書いたファイルを置く

-. check.rb を実行

-. results.txt にエラーのファイル一覧が表示される。
