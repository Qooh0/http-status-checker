http-status-checker
======================

upper ruby2.0

no warranty

Usage
-------

1. written urls in target-urls/hoge.txt , target-urls/some.txt ... 

2. run `check.rb` 

3. check result `cat results.txt` 

使い方
-----------

1. target-urls の中のファイルは適当に読むので、そこに調べたい url を書いたファイルを置く

2. check.rb を実行

3. results.txt にエラーのファイル一覧が表示される。
