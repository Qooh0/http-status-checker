http-status-checker
======================

upper ruby2.0

no warranty

Usage
-------

1. mkdir target-urls 

2. written urls in target-urls/hoge.txt , target-urls/some.txt ... 

3. run `check.rb` 

4. check result `cat results.txt` 

使い方
-----------

1. target-urls というディレクトリを作成する

2. そのディレクトリとの中のファイルは適当に読むので、そこに調べたい url を書いたファイルを置く

3. check.rb を実行

4. results.txt にエラーのファイル一覧が表示される。
